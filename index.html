HomeIntel

HomeIntel is a lightweight browser extension built for OSINT workflows. Instead of manually opening multiple resources, it automates address lookups by launching several open-source sites at once. This makes it easier and faster to cross-reference addresses across trusted sources.
🚀 Multi-Site Search – Instantly opens several open-source lookup tools in new tabs
🕵️ Investigator-Friendly – Speeds up OSINT and geolocation research
🌐 Simple & Lightweight – Developed with HTML and JavaScript
📌 Practical Utility – Great for open-source investigations, fact-checking, or situational awareness

PandoraBox

PandoraBox is a Python-based cryptography and encoding toolkit designed for learning, experimentation, and practical application of common data transformation techniques. It includes:
🔐 Base64 Encoding/Decoding – Convert text to and from Base64 format
🏛️ Caesar Cipher (+3) – Apply a classic Caesar shift cipher for simple encryption
🔄 Text Reversal – Reverse any input string
🔒 SHA-256 Hashing – Generate secure hash digests of text inputs
🧠 MD5 and SHA-256 Brute-Forcers – Attempt to recover original text by brute-forcing hash values using dictionary attacks
